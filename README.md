# ⚖️ Offer 選擇障礙終結者 (Offer Decision Helper)

別再擲筊、抽籤、問神明了！用科學加權演算法，挖掘你內心深處最真實的選擇。這是一個專門為面臨複數 Offer 抉擇的工作者設計的**決策輔助工具**。

透過直觀的現代 Web 介面，本工具引導你定義個人的職涯偏好權重，並為各家公司打分，最後透過加權演算法與資料視覺化（雷達圖、直條圖），幫你找出最理想的職涯首選。

---

## ✨ 核心特色

- 🎨 **現代質感 UI/UX**：採用 Glassmorphism 毛玻璃視覺設計與深色模式配色，提供舒適、尊榮的視覺體驗。
- 🏢 **動態公司名單管理**：支援即時新增公司或點選 `🗑️` 刪除，並能即時保存修改後的公司名稱。
- 🛠️ **高度自訂評估維度**：
  - 預設提供常見指標：`💰 薪資福利`、`📈 技術與職涯成長`、`🏡 WLB與彈性`、`🤝 團隊氛圍與主管`。
  - 支援動態新增自訂評估項目（如：出差頻率、公司穩定度）或刪除不需要的維度。
- 📊 **雙維度資料視覺化**：
  - **決策維度雷達圖**：快速比對各個 Offer 在各領域的覆蓋面積，面積越大代表該 Offer 越符合你的理想標準。
  - **分項對比直條圖**：橫向對比不同公司在單一維度上的滿意度得分。
- 🚨 **糾結與伯仲警報**：當前兩名 Offer 加權滿意度差距在 4% 以內時，系統會觸發警報提示，並給出科學的決策微調建議。
- 📥 **評估報告匯出**：支援一鍵下載完整評估數據為 CSV 格式。

## 🚀 快速開始

### 1. 安裝環境與相依套件

本專案基於 **Python 3** 與 **Streamlit** 開發。請先確保已安裝 Python 3.10+，並在專案根目錄下執行以下步驟：

```bash
# 建立並啟用虛擬環境 (Windows)
python -m venv .venv
.venv\Scripts\activate

# 建立並啟用虛擬環境 (macOS/Linux)
python3 -m venv .venv
source .venv/bin/activate

# 安裝所需套件
pip install streamlit pandas plotly numpy
```

### 2. 啟動應用程式

安裝完成後，執行以下指令啟動 Streamlit 伺服器：

```bash
streamlit run app.py
```

啟動後，瀏覽器會自動開啟，或者你可以手動瀏覽：
👉 [http://localhost:8501](http://localhost:8501)

## 📝 決策步驟指引

1. **Step 1: 填入你的 Offer 資訊**  
  點選 `➕ 新增一間公司` 增加比較對象，或者點擊 `🗑️` 刪除，並在輸入框中修改公司名稱。
2. **Step 2: 設定指標權重 (1-10 分)**  
  數字越大代表你越看重這個項目。如果需要，可在「🛠️ 自訂評估維度與說明」中新增屬於你自己的特殊指標。
3. **Step 3: 為各公司打分 (1-10 分)**  
  點開各公司的評估卡片，針對各指標給予滿意度評分（1 分最差，10 分完美）。
4. **Step 4: 閱讀診斷報告**  
  右側會即時渲染出「加權總分」、「滿意度百分比 %」以及各項圖表。請特別留意雷達圖的覆蓋範圍與下方的決策大師小建議。
